Here is what Mu-Gen does for you!

Mu-Gen uses LSTM Architecture

Mu-Gen uses modernistic AI algorithm 'LSTM' to generate melodies inspired from different regional folk songs from all over the world. LSTM is a special kind of Recurrent Neural network.
RNN is a type of Neural Network (NN) where the output from previous step are fed as input to the current step. In other words, RNN is a generalization of feed-forward neural network that has an internal memory. RNNs are designed to recognize a data’s sequential characteristics and use patterns to predict the next likely scenario.

About ESAC Dataset

  1. Over 20k songs from all over the world. The project started in 1994
  2. ESAC stands for “Essen Associative Code and Folk song database”
  3. Region wise folk songs are available from across many continents (mostly european).

Data files downloaded in kern format (.KRN).
Folk songs from the region of Austria, China, Czech Republic, Germany, Hungary, Lorraine region of France, Netherlands, Poland, Switzerland, Yugoslavia taken.

About Kern

Kern- encoding for symbolic representation. Kern was developed in the 80s, in the context of a software (library) Humdrum. Humdrum is a library where you can analyse symbolic music. Kern is the encoding (music representation) used by the Humdrum software package.
The kern representation is used to represent basic or core information for music. The kern scheme can be used to encode pitch and duration, plus other common score-related information.
